# Herroy Analytics

Data analytics projects focused on business insights, strategy, and hospitality performance.

## Project 1: Bar Revenue Analysis

This project explores simulated bar sales data to identify:

- Total revenue
- Revenue by product
- Top-performing product
- Best revenue day
- Basic business recommendations

## Tools Used

- Python
- pandas
- matplotlib
- Jupyter Notebook
- Git / GitHub

## Business Context

The objective is to demonstrate how simple transactional data can be transformed into practical business insight for hospitality environments such as bars, restaurants, and hotel F&B operations.
